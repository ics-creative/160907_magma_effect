# マグマエフェクト

three.jsを使ったマグマフェクトサンプルです。
記事『[エフェクト作成入門講座 マグマエフェクト作成 three.js編](https://ics.media/entry/13973/)』で解説しています。

このプロジェクトはViteを使用しています。

## Setup

```
npm install
```

## Run development server

```
npm run dev
```

## Build for production

```
npm run build
```

## Preview production build

```
npm run preview
```
